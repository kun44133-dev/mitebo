# App 修改记录

本文件用于持续记录 App 的每次功能、界面和稳定性修改。最新记录放在最上方。

## 2026-07-18

### 打包版本 1.0.78

- `versionCode` 更新为 `78`，`versionName` 和登录页版本号更新为 `1.0.78`。
- 本次安装包包含告警模具状态锁定、离线模具提示点颜色调整、语音播报测试入口融合。

### 告警模具状态锁定

- 告警列表中的“在线模具/离线模具”状态改为按告警首次进入 App 时的模具状态锁定。
- 在线模具产生的告警后续仍显示为“在线模具”，离线模具产生的告警后续仍显示为“离线模具”，不再跟随模具当前状态变化。
- 锁定状态按登录账号隔离保存，避免切换账号后串用其他账号的告警状态。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

### 离线模具提示点颜色

- 离线模具页面顶部提示条左侧圆点改为使用离线状态胶囊背景色。
- 在线模具和网关管理页面仍保持原有绿色提示点。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

### 语音播报测试入口融合

- 设置页去除单独“测试语音”按钮。
- 打开“语音播报”开关时自动播报测试语音，同时关闭普通报警声音。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

## 2026-07-17

### 打包版本 1.0.77

- `versionCode` 更新为 `77`，`versionName` 和登录页版本号更新为 `1.0.77`。
- 本次安装包包含锁屏后台语音播报、TTS 唤醒锁和删除告警后数量即时刷新修复。

### 锁屏后台语音播报

- 后台告警轮询服务新增 TTS 播报能力，锁屏状态下发现新告警时由前台服务直接播报，不再依赖打开 App 界面。
- 告警语音播报时短暂持有 `PARTIAL_WAKE_LOCK`，降低锁屏后 CPU 休眠导致 TTS 不播报的概率。
- 如果后台 TTS 初始化失败，会自动关闭语音播报并切回普通报警声音兜底。
- 影响文件：`app/src/main/AndroidManifest.xml`、`app/src/main/java/cn/mitebo/iot/AlarmMonitorService.java`。

### 告警删除后数量即时刷新

- 删除告警信息成功后，立即本地扣减当前日期告警数量、历史告警数量和未消除角标数量。
- 删除后下一次列表刷新允许数量下降，避免“只增不减”的异步保护导致顶部数量不及时更新。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

## 2026-07-16

- 设置页“测试语音”和调试测试语音仍保持单次播报，方便快速验证 TTS 是否可用。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

### 打包版本 1.0.75

- 优化告警语音播报文本，避免系统 TTS 将模具编号、传感器编号逐字母朗读。
- 模具编号播报改为提取末尾短编号，传感器名称优先播报中文描述；无中文名称时只播报传感器尾号。

### 打包版本 1.0.74

- `versionCode` 更新为 `74`，`versionName` 和登录页版本号更新为 `1.0.74`。

### 语音播报测试与兜底

- 设置页新增“测试语音”按钮，可直接测试当前手机是否能播放中文 TTS。
- 开启“语音播报”后会自动播报“语音播报测试成功”，用于确认语音引擎可用。
- 如果 TTS 初始化失败、中文语音不可用或播报失败，会提示“当前手机语音引擎不可用，已切回报警声音”，并自动关闭语音播报、打开普通报警声音。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

### 告警语音播报开关

- 设置页新增“语音播报”推拉开关，开启后使用系统 TTS 播报告警内容。
- 语音播报开启时会自动关闭“报警声音”，再次手动开启报警声音时会关闭语音播报，避免铃声和语音同时播放。
- 新告警播报格式为“模具 C155，下模1区压力低，报警压力 77.6 bar”，优先读取模具编号、传感器名称和触发压力。
- 语音播报触发时保留震动提醒，退出页面、停用告警或销警后会停止当前语音。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。
