# App 修改记录

本文件用于持续记录 App 的每次功能、界面和稳定性修改。最新记录放在最上方。

## 2026-07-01

### 后台告警轮询稳定性

- 后台告警请求改为上一轮完成后再安排下一轮，避免网络较慢时多个轮询请求重叠。
- 增加请求运行状态和轮询代次校验，切换账号或停止服务后丢弃旧请求结果。
- 接口请求失败时保留现有告警状态，不再误清空角标和正在告警状态。
- 后台监控关闭、账号失效、版本过期或前台服务超时时，统一停止轮询、铃声和通知。
- 后台告警服务只拉取告警列表，不刷新设备压力，避免影响静止压力采集。
- 影响文件：`app/src/main/java/cn/mitebo/iot/AlarmMonitorService.java`。

### 模具状态与静止压力锁定

- 区分模具列表状态和设备内嵌模具状态，避免把设备记录启用状态误判为模具生产状态。
- 无法确认新状态时延用上一次状态，避免短暂缺少字段导致在线模具误下线。
- 静止压力锁定状态和下一轮待采集状态写入本地存储，App 重启后继续保持。
- 只有模具经历在线生产并再次下线后，才允许开启下一轮静止压力采集。
- 刷新接口状态不明确时不再用实时动态压力覆盖已锁定的静止压力。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

### 设备与网关界面

- 设备卡片增加 `MAC：` 标签，并将在线状态胶囊放到 MAC 地址同一行。
- 收窄设备卡片右侧压力区域，为 MAC 地址和状态留出更多空间。
- 网关卡片操作按钮去除文字图标、缩小宽度并增加中间间距。
- 登录账号下拉箭头适当放大；设置页压力单位说明改为“切换压力单位”。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

### 网关定位开关

- 网关定位状态从手动输入 `0/1` 改为“关闭/开启”下拉选择。
- 保存时按后端需要提交整数 `0/1`，并兼容接口返回的数字、布尔值和字符串状态。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

### 打包版本 1.0.52

- 本次安装包包含模具信息卡“编号：”“名称：”标签调整。

### 模具信息卡标签

- 模具信息卡的两行标题调整为“编号：”和“名称：”。
- 同步加宽标签区域，避免冒号遮挡编号或名称内容，并保持两行数值左对齐。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

### 版本

- APK 版本更新为 `1.0.51`，`versionCode` 更新为 `51`。

### 静止压力采集时间调整

- 将传感器静止压力的连续稳定判定时间从 10 秒调整为 3 分钟。
- 3 分钟内压力再次产生超出稳定容差的波动时，会重新开始计时。
- 采集成功后继续锁定静止压力，直到模具再次上线并进入下一次下线采集周期。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。

### 网关模具防重复绑定

- 网关更换模具或修改网关信息时，候选列表不再显示已被其他网关绑定的模具。
- 当前网关已经绑定的模具仍会保留在列表中，便于查看和正常更换。
- 网关绑定关系按分页完整读取；绑定关系加载失败时保留原候选列表，避免误清空。
- 设备页面的模具绑定列表不受此规则影响。
- 影响文件：`app/src/main/java/cn/mitebo/iot/MainActivity.java`。
